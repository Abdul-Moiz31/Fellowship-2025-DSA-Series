## 📘 7. Fibonacci Number (Nth Term)

### 📝 Description
Find the **Nth term** of the Fibonacci sequence using recursion.

Definition:
fib(0) = 0
fib(1) = 1
fib(n) = fib(n-1) + fib(n-2)

### 🔢 Input
A single integer `n`  
**Constraints:** `0 ≤ n ≤ 30`

### 📤 Output
An integer representing the `n`th Fibonacci number.

### 📌 Example
Input: 6
Output: 8
