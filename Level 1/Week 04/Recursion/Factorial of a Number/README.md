## 📘 Factorial of a Number

### 📝 Description
Calculate the **factorial of N** using recursion.  
Definition:  
`N! = N × (N - 1) × ... × 1`  
Base Case: `0! = 1`

### 🔢 Input
A single integer `N`  
**Constraints:** `0 ≤ N ≤ 20`

### 📤 Output
An integer representing `N!`

### 📌 Example
Input: 4
Output: 24
